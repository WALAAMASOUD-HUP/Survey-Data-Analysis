📊 Professional Survey Data Analysis — Power BI
This project presents an interactive dashboard built using Power BI to analyze insights from a professional survey dataset. The report visualizes key metrics related to salary, job titles, demographics, satisfaction levels, and programming language preferences.
________________________________________
🔍 Overview
The dashboard provides a clear, data-driven view of the survey results through multiple visualizations, including KPIs, maps, tree maps, donut charts, and bar charts.
It helps stakeholders quickly understand workforce trends and identify patterns across countries, gender, job titles, and career satisfaction.

________________________________________
📈 Key Insights Displayed
1. Key Performance Indicators (KPIs)
•	Total Survey Count
•	Average Age of Respondents
2. Salary Analysis
•	Average Salary by Country (Map visualization)
•	Average Salary by Job Title
•	Average Salary by Gender (Male vs Female)
3. Satisfaction Metrics
•	Happiness with Work/Life Balance
•	Happiness with Salary
4. Geographic & Category Analysis
•	Country Distribution of Survey Responses (TreeMap)
•	Favorite Programming Languages (Bar Chart)
________________________________________
🛠️ Tools & Techniques
•	Power BI Desktop
•	Data modeling with Power Query
•	DAX Measures for KPIs
•	Custom visualization formatting
•	Map, donut chart, tree map, and bar chart visuals
________________________________________
📁 Dataset
The dataset includes anonymized survey information such as:
•	Age
•	Country
•	Job Title
•	Salary
•	Gender
•	Satisfaction scores
•	Favorite programming languages
